# Forgotten Society — Audio MITM (nekto.me)

Двухсторонний MITM на голосовой чат nekto.me.

```
Собеседник 1  <—WebRTC—>  Клиент 1  <—mixer—>  ВЫ  <—mixer—>  Клиент 2  <—WebRTC—>  Собеседник 2
```

## Запуск

```bash
cd forgotten-society
npm install
node tools/get-tokens.js 2     # получить два guest userId
npm start                      # http://localhost:8000
```

В UI:
1. Главная → "Начать"
2. Если токенов нет — экран "Токены": вставь два разных userId, Сохранить.
3. Разреши микрофон (он по дефолту замьючен — твой голос НЕ уходит к собеседникам).
4. Оба клиента регистрируются → стартует поиск → как только оба нашли собеседников, в записи слышны оба голоса, а собеседники слышат друг друга через тебя.
5. "💾 Скачать диалог" — выгрузка webm/opus.

## Файлы
- `server.js` — express, статика + прокси `/register` к nekto.me fingerprint API
- `public/index.js` — основная логика (исправленный исходник)
- `public/lib/fp.js` — обёртка над FingerprintJS v4 OSS
- `tools/get-tokens.js` — регистрация guest-аккаунтов

## Ключевой фикс относительно исходника
В оригинале `Client.peer.addTrack` отправлял `this.stream`, а это **входящий** поток от собеседника. Теперь шлём `this.input` = микшированный output соседнего клиента → собеседник 1 слышит собеседника 2 и наоборот.
